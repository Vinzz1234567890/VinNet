## 🚀 Core Features

* **Shell 3 Architecture:** Lightweight post-boot script execution.
* **Aggressive Network Tuning:** Comprehensive optimization from framework level, vendor level, until kernel level.
* **Latency Mitigation:** Reduces Wi-Fi and mobile data power-saving sleep cycles, improve stability.
* **Zero Overhead:** Perform comprehensive optimization without causing system instability.

## ⚙️ Prerequisites

> **Warning**
> Remove all conflicting network modules before proceeding.

* Rooted Android Environment.
* Magisk (Stable, Canary, Alpha, Beta, Delta/Kitsune), KernelSU / KernelSU Next / KowSU / SukiSU-Ultra, APatch / FolkPatch.

## 📦 Instructions

<details>
<summary><b>Installation Instructions</b></summary>

1. Download `VinNet.zip` from [Releases](https://github.com/Vinzz1234567890/VinNet/releases).
2. Open root manager.
3. Navigate to **Modules** > **Install from storage**.
4. Flash `VinNet.zip`.
5. Reboot.

</details>

<details>
<summary><b>Uninstallation Instructions</b></summary>

1. Open root manager.
2. Navigate to **Modules**.
3. Select **Remove** on VinNet.
4. Reboot.

</details>
